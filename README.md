## ğŸ”§ Customization

### Colors
All colors are defined in `assets/css/variables.css` using CSS variables. Update them in the `:root` section to change the entire color scheme.

### Fonts
The project uses system fonts via `font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif`. Update in `assets/css/style.css` to use custom fonts.

### Bootstrap
Bootstrap 4 is loaded from CDN. For offline use, download Bootstrap and update the link tags.
